Базовая структура

ОБЯЗАТЕЛЬНЫЕ:
-functions              папка для функций
-sound                  папка для звуков
init.sqf                первый выполняемый скрипт миссии
setup.sqf               скрипт для установки начальных переменных, констант, состояний и т.д.
createDiary.sqf         скрипт для создания брифинга
description.ext         импорт новых звуков, твики конфигов


ОПЦИНОНАЛЬНЫЕ
missionFlow.sqf - файл для контроля процесса выполнения миссии. Включает вызовы доп функций, сприктов для триггеров и логики миссии
