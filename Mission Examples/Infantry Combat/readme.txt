Пример миссии для пехотного боя

Mission Flow
1. Перегруппируйтесь с солдатами
2. Пополните боезапас
3. Доложите о готовности штабу
4. Захватите позицию противника
5. Удерживайте позицию, отразите контратаку
6. Отразите вторую контратаку
7. После отражения контратаки грузовик с подкреплением начинает движение
8. Финальный диалог с штабом и завершение миссии

Особенности миссии
Система здоровья из Call Of Duty
Система реанимации ботов (Игрока реанимировать нельзя)
Бесконечный боезапас у ИИ в отряде игрока
Контроль зоны операции с убийством игрока его же солдатами
